一 夏日 初程

  1. 夏日炎炎当属出去玩才嗨!

7 月 3 日，暑假的开端，香蕉中学洋溢在暑假来临的无限欢乐之中而其中一位却似与此番景象格格不入。
高一 18 班最后一排靠窗角落，长告正坐在座位上眺望窗外沉思着，不时转头看向何染，她正在和其他女生商量着暑假去哪里玩。

“你们真是期待值拉满了啊…”此刻水母内心既兴奋又失落。兴奋的是可以愉快地爽打游戏了，失落的是回来之后分班，自己选理，何染选文。二人不可能在同一个班了。趁这会能看几眼是几眼，又或者，暑假直接大胆点了?…

长告又望向窗外，耳朵却仔细聆听着女生谈话。

“要不我们去海南玩？”何染问道。

海南？这 TM 就是她向往的地方？小本子先记下来。水母这样想着。

“嗯，这也不错。游泳，晒太阳，喝椰汁…”女生甲道。

“对啊对啊，不错吧？”何染兴奋地应和。

游泳？也确实。夏天就该游泳。先要准备泳衣吧… OIOIOI！不是水母你脸红啥？水母内心想着，脸已经涨得通红。心脏飞速跳动着，面朝窗户，他已经极力克制住转移视线了。

大脑中两个小人正自由搏击，小长告说，你是香中高中生了，自矜一点，保持人性！小水母上去就是一拳，你 TM 装啥呢，好色是人的天性，现在不看，等分班见面也少了吧！二人争斗着，小水母最终获得控制权。

没事，偷偷看又没人注意，私密马赛了，小长告。长告猛地一转头正想着打开自瞄，迎面撞上了一个人的脸。

“WC，你怎么在这!?”水母吓得习惯性后倒，看着眼前的人。

“没事啊，过来看看，一进来就看见你躲在角落里偷偷傻笑，猥琐死了，想啥呢老弟（你想啥我还能不知道吗）”何惠双手交叉于胸前，询问长告。

“我…还能想啥，放假了我高兴。怎么着，不爽是吧？”长告有丝丝紧张，目光却不时地瞟向何染。“话说，你来我们班做什么？”

这点小动作怎么可能逃得过何惠的眼睛呢。“不用猜也知道你那点猥琐的小心思，不用藏着掖着老弟，没什么丢人的，我来就是和何染准备放学一起回家的。”说完何惠转头欲走。

“哦对了，你的计划，趁早行动吧。”何惠做出一个阴险狡诈的笑容。“暑假，可是会很快过去的哦。”

“什…什么计划？”长告还想装一把。

“姐姐我可是看好你的哦。”

“喂，什么看好？你说清楚啊喂！别走！”

二 水母也想被邀请

“啊~~好无聊…”，长告躺在床上，用手遮住脸，不愿太阳直射他的眼睛，嘴上却念个不停。

暑假已经被他浪费了七天，这七天，作业完成率 0%，游戏段位提升率 91%，计划完成率 0%。不能说进展细微，只能说一事无成。

“什么时候，暑假变得这么无聊了…”

突然间，放在床头的手机连叮好几声，是短信！水母也有一丝丝激动了，急忙打开手机。

  1. 短信 10086
    亲爱的 139****0910 用户，您的账户花费已欠费，若不按时缴费，将无法呼出…
  2. 短信 广东 91 传媒有限公司：
    看到第二条，长告急地狂点右上角 “-” 号，删除了短信，又打开巨信查看余额，0.91 元，昨天 91 刚冲 191，没钱充话费了…

长告正想打巨信通话给 JC 爆点金币充话费，好友列表 “潘岳坤” 发来消息。
水母，你去不去？

水母纳闷，什么去不去的，问了一句。

不是吧水母，何染不是邀请了几个人一起去玩密室逃脱吗，莫非你没被邀请？不是吧哥们？

水母呆呆地看着手机屏幕。与何染要好的，班上除了水岳二人，还有几个女生。男生中也就他和岳坤关系要好，可是这次他却未收到邀请，水母如遭晴天霹雳，手机滑落在地上。

为什么我没收到邀请，莫非她讨厌我了？或者，她从不视我为朋友…
又或者她也喜欢我，故意躲着我…
（蚌埠住了）

无数想法涌入水母脑中，但最终还是被压垮倒在了床上。

三 误邀与偶遇

1. 欠费的勇气

“为什么我没被邀请……”
长告把脸埋进枕头里，闷得像被关在密室的密室里。手机在地上屏幕朝下静静躺着，像给他判了个“失联缓刑”。

小长告（理性小人）叹气：“你要是现在去补作业，至少还能保住年级前一百的体面。”
小水母（感性小人）拽起他：“体面有啥用？爱情来过你家门口，你连猫眼都不看一眼？”

门外突然“咔哒”一声——是水姐投喂的冰西瓜。长告抬头，盯着瓜籽一排排像密码。
密码……密室…… 大脑里某根断线啪地接回去了。

他翻身下床，摸到手机，连上家里 Wi-Fi，刚要发消息，头像那边“潘岳坤（在线）”先炸了：

岳坤：“水母，你去不去？”
长告：“去哪？”
岳坤：“就你暗恋对象组织的那个密室啊！你没被拉？不会吧哥们。”

长告指尖一滞，屏幕倒映出他乱七八糟的刘海。
没被拉，像一把小刀，恰好插在犹豫和自尊的夹缝里。

这时，另一个对话框跳到顶端：何惠。

何惠：“老弟，给你两条路：
A. 躺平等天黑；
B. 来‘椰风广场’四层 402 密室店，下午三点。
PS：欠费？一楼超市充电话卡，右转第一家。”

她怎么知道他欠费？哦，对了，昨天在走廊她顺走了他充满 Bug 的“计划表”。
被看穿的感觉不体面，但被指路的感觉……像被人从水里捞起来。

小水母拍桌：“B，选 B！”
小长告磨磨蹭蹭：“至少先洗个头。”

2. 椰风广场的剧本杀

椰风广场像一艘停在城里晒太阳的白色邮轮。长告在超市充了 50 元话费，结账小票上印着“夏日十足”。他看着那四个字，忽然就不那么心虚了。

密室店门口，一条蓝色的霓虹字写着：《深海 42 米》。
前台姑娘把表格往前一推：“今天 402 房人满一个名额刚好空出来。”

长告张了张嘴，还没来得及说“太巧了吧”，身后就传来熟悉的笑声：

“猥琐男，来啦？”

他回头——何惠双手插兜，像个押送犯人去体验生活的警察。她身后晃过一缕发梢，是何染，正低头系鞋带，没看他。

“我……”长告发现所有准备好的开场白都塞在喉结下面。“我就是路过。”

“路过到正好少一人，妙啊。”何惠眯眼，“进去记得呼吸。”

小长告：“注意礼仪，先跟所有人 say hi。”
小水母：“注意她的鞋带，找话题！”

长告咽了下口水，走到何染旁边。
“需要帮忙吗？”
“……嗯？哦，谢了。”何染站起身，眼睛弯了下，“你也来玩？”
“路过。”
“哈哈，行吧，路过的就跟紧点。”

他忽然觉得“路过”这两个字听起来不那么寒酸了，像个编进剧本里的暗号。

3. 深海 42 米

密室里一股淡淡的消毒水味，墙上是发蓝的灯条，地面有浅浅的波纹投影。海难主题：寻找“黑匣子”，解开“沉船航线”。

第一关是一面拼图墙，碎片像贝壳。女生甲刚抱怨形状太多，长告就本能地按色块分区、按卡槽角度归类。
“你以前玩过？”何染看他手起片落。
“打游戏打多了，都是 UI 交互。”
“那你牛。”她竖了竖大拇指。

第二关是密码锁。提示卡写着：“母亲之水，三相一体。”
“水母？”长告脑子里先蹦出自己外号，紧接着是教学楼墙上的化学海报——H₂O、固液气。
他试了“H2O”，“solidliquidgas”，“012”……统统不对。

何染半蹲在另一侧，抬眼提醒：“三相点的温度？”
“0.01℃！”长告眼睛一亮，拨盘“0-0-1”。
“咔哒。”

第三关是紫外灯扫描。墙面上浮出一行字：RAN / COCONUT / 18。
女生甲：“COCONUT 是椰子？”
长告心口一紧——海南、椰汁、18 班——这些词像细小的气泡往上冒。他想说点什么，又怕露怯。

何染先一步伸手按下“18”号壁龛，里面弹出一枚小小的“黑匣子”。她回头冲大家笑：“破案！”

尾关是“逃生梯”。灯光从幽蓝变暖白，背景音里的海浪退下去。门开的一瞬间，长告听见自己的心跳从“扑通扑通”慢慢放缓。

他偷瞄何染，恰好对上她的视线。
“路过得不错。”她说。

4. 奶茶与分班

店门外的走廊吹来空调风，奶茶杯壁起了雾。
大家七嘴八舌地复盘。女生甲夸“小水母（长告）这手速像打天梯”，潘岳坤在旁边疯狂对嘴型“发！挥！稳！”。

何惠趴在栏杆上看手机，突然抬头：“有人要坐公交回家，有人要去隔壁泳装店。”
女生们哄笑着推搡她，她被挤回栏杆，“开玩笑，别打我。”

何染吸了一口奶茶，转向长告：“你暑假有计划吗？”
“有。”长告脱口而出，连他自己都被吓了一下。
“说来听听？”
“先学会不欠费。”他举杯示意。众人一顿，随后笑成一串。

笑声落下时，走廊安静了半拍。何染收敛了一点点笑意：
“分班之后，可能不常见面了。”
“嗯。”
“那……先把想做的事做了吧。”

她把手机亮给他看，是一张便签：

《夏日清单》

  1. 游泳。
  2. 太阳。
  3. 椰汁。
  4. 一部很烂但很爽的电影。
  5. 给喜欢的人打一次电话。

小水母在他脑壳里按了导弹红钮：“RUSH！”
小长告还想套个确认框：“Are you sure?”

长告深吸一口气：“第一条，我可以当教练。”
“你会？”
“会想尽办法学会。”他笑，笑得有点笨拙，却不再虚。
“那成交。”何染伸出手，“下周二，桃溪游泳馆。”

他们轻轻击掌。掌心的水汽和奶茶的冰一起被空气带走，剩下一个不会过期的约定。

5. 姐姐的消息

少顷，众人散去。长告和潘岳坤慢吞吞下楼。
岳坤把他拽到拐角处：“你小子可以啊，路过就路到她夏日清单上去了？”
长告挠头，脸上那点可疑的红终于藏不住。

手机“叮”了一声，是何惠发来的：

“B 选得漂亮。接下来的作战目标给你三条线索：
【泳衣】、【太阳】、【椰子】。
每完成一条，发我汇报。超时会被我骂。”

底下跟了一张打了马赛克的照片：
蓝色泳池的一角，编号“18”。

长告仰头，看见商场中庭的天窗被夕阳涂成金色。他忽然明白了“夏日十足”的意思：不是热，是热闹；不是躁，是走。

小长告把一本新计划册啪地掀开：“目标一：学会憋气 30 秒。”
小水母在旁边涂了一行更大的字：“目标零：别怂。”
